# basic

# vscode 自动编译 .ts 文件

  1. 在根目录下 运行tsc --init 生成配置文件
  2. 在配置文件 设置 outDir 编译后 js 文件的输出目录
  3. 点击vscode 菜单 > Terminal > Run Task > typescript > tsc:watch - tsconfig.json

设置完成后就可以自动生成代码

# 数据类型

typescript 中为了使编写的代码更规范, 更有利于维护, 增加了类型校验

写 ts 代码必须指定类型

  • 布尔类型 (boolean)
  • 数字类型 (number)
  • 字符串类型 (string)
  • 数组类型 (array)
  • 元组类型 (tuple)
  • 枚举类型 (enum)
  • 任意类型 (any)
  • null 和 undefined
  • void 类型
  • never 类型

# 枚举类型(enum)

在程序中用自然语言中相应含义的单词来代表某一状态,使程序容易阅读和理解,也就是说,事先考虑到某一个变量可能取的值, 尽量用自然语言中含义清楚的单词来表示它的每一个值. 这种方法称为枚举, 用这种方法的类型称枚举类型.

// enum Flag { success, error}
enum Flag { success = 1, error = 2}

let s: Flag = Flag.success

console.log(s)   // 1

enum Color { blue, red, 'orange'}  // 加不加引号 一样
let c: Color = Color.red
console.log(c)   // 1

//默认一项项的值是从0开始 递增 1   如果有赋值, 则后面的值由赋值后开始 递增 1

  name: string;    // 属性, 前面省略了 public 关键字

  }

  }

  }

var p = new Person("张三")

class Student extends Person {
  constructor(name:string) {
    super(name);              // 初始化父类的构造函数
  }

  run():void{
    //子类与父类  有同样的方法,  会覆盖掉父类的方法
  }
}

# 修饰符

ts定义属性的有三种修饰符

  • public : 公有 在类里面, 子类, 类外面都可以访问

  • protected : 保护类型 在类里面, 子类里面可以访问, 在类外部无法访问

  • private : 私有 在类里面可以访问, 子类, 类外部都无法访问

属性没写名修饰符, 则默认是public

# 抽象类

抽象类,它是提供其它类继承的基类, 不能直接被实例化.

用abstract关键字定义抽象类和抽象方法, 抽象类中的抽象方法不包含具体的实现

抽象方法只能放在抽象类里面

抽象类和抽象方法用来定义标准, 继续它的子类必须实现抽象方法

abstract class Animal {
  abstract eat():any;
}
// 这个抽象能不能实例化

class Dog extends Animal {
  construnctor(){

  }
  eat() {
    // 实现抽象类的方法
  }
}

# 接口

接口的作用: 在面向对象的编程中, 接口是一种规范定义, 它定义了行为和动作的规范,在程序设 计里,接口起到一种限制规范的作用. 接口只定义规范,不关心具体的实现.ts中的接口类似java 同时还增加了更灵活的接口类型,包括属性,函数,可索引和类等.

属性接口 对json的约束


interface FullName {
  firstName: string;
  secondName?: string;  // ? 表示可选
}

function printName(name: FullName) {

}

printName({
  firstName: 'chen',
  secondName: 'Kevin',    //这样写,必须和 interface 定义的一样
})

var obj = {
  age: 20,
  firstName: 'chen',
  secondName: '三'
}
printName(obj)     //传入的参数必须包含 firstName 和 secondName

函数类型接口 对方法传入的参数 以及返回值进行约束

interface encrypt {
  (key:string, value: string):string;
}

var md5:encrypt = function(key:string, value:string): string {

}

//另一种定义函数
var md5: (key:string, value:string) => string = function(key:string, value:string): string {

}

可索引接口,数组,对象的约束 (不常用)
