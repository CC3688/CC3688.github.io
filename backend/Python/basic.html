# 基础知识

python是一种解释型的语言

python2 默认编码方式是ascii码,

而python3默认编码方式是utf-8

所以最好在 行首行添加

#-*- encoding:utf-8 -*-

# 变量

变量: 就是将一些运算的中间结果暂时存到内存,以便后续代码调用

1, 必须由数字, 字母, 下划线任意组合, 且不能数字开头 2, 不用是python中的关键字 3, 变量具有可描述性。 4, 不能是中文。

tips: 多个单词, 推荐, 用 _ 来连接

# 常量

没有真正的常量, 是用大写来约定,它是一个常量,请不要改它而已.

# input

用户交互, 在终端,接收用户的输入

注意它接收的 数据类型全部是str


name = input('请输入你的名字:')  # 参数,是显示在终端的提醒文字
age = input('请输入你的年龄:')

print('我的名字是:', name, '我今年', age, '岁了')

# if

# 第一种
if 4 > 5: 
  print('为真则执行这里')

# 第二种
if 4 > 5:
  print('为真则执行这里')
else:
  print('为假则执行这里')

# 多选
if num == '1':
  print('为真则执行这')
elif num == '2':
  print('为真则执行这')  
elif num == '3':
  print('为真则执行这')  
elif num == '4':
  print('为真则执行这')  
else: 
  print('以上条件没一个为真,才执行这')

# 注意, 这种elif 当一个条件成立后, 后续的代码就不会执行了, 要注意,条件出现的先后顺序

# 循环

# while

while flag: 
  print('while后接着的flag
  为真,就执行,要注意,避免死循环')

count = 1
while count <= 100:
  print(count)
  count = count + 1

# 会一直打印 1 2 3 ....  100

continue

跳过这次循环, 即 continue后的代码就不执行了,

回到循环的开始去,重新判断flag是否为真,真的则进入循环,假就结束循环.

break 跳出循环, break后面的代码不执行,并跳出循环
