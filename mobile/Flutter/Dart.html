# 常量

永远不变的量,可以用 final const 来声明

  • final

    • final name = "Kevin"
    • 用于修饰变量，表示单赋值（single-assignment），使用final修饰的变量必须进行初始化，一旦被赋值之后，不能够再次被赋值,否则编译会报错。跟java的区别是：final不能用来修饰类和方法。

  • const

    • const PI ＝ "3.14"
    • 表示对象的状态完全可以在编译期间确定，并且完全是不可变的。
    • const变量不会在内存中重复创建，如果表达式被调用了多次，则重用之前创建好的常量。
    • const构造函数的前提是成员变量必须都是用final修饰的。const单例多例可以如下实现

  • static

    • 用于修饰成员变量，这个变量是属于类的，而不是属于对象的，除此之外没有其他别的含义了，static修饰符的使用方式和Java类似。 static变量直到运行期被使用时才会实例化。

# 命名参数

void main(){
  var result = findVolume(2, breadth: 3, height: 10);
  print(result); // 50

  var res2 = findVolume(2, height: 10, breadth: 3);
  print(res2); // 50
}

int findVolume(int length, {int breadth, int height}) {
  return (length + breadth) * height;
}
  • 传参数, 位置不再重要了,可以不按位置传, 上例,第一个参数是位置参数,它还是要按位置来传

# 系统错误处理

void main() {
  int result = 12~/0;
  print(result);
}
  • ~ 表示,除后结果为 int 类型
  • 除数不能为 0 , 系统会抛出一个错误, 而停止整个应用的运行
  try {
    int result = 12~/0;
    print(result);
  } on IntegerDivisionByZeroException {
    print("除数不能为0");
  }


  try {
    int result = 12~/0;
    print(result);
  } catch(e) {
    print(e);
  }


  try {
    int result = 12~/0;
    print(result);
  } catch(e, s) {
    print(e);
    print(s);
  }


    try {
    int result = 12~/0;
    print(result);
  } catch(e) {
    print(e);
  } finally{
    print("finally")
  }
  • 预知到错误的类型, 可以用on 具体的错误类型
  • 无法预知,就用 catch(e) e是具体的错误类型
  • s 是错误栈, 即可以用它来找到哪里出错了,错了什么
  • finally 里的代码,无论是否有错误发生,它都会执行

void main() {

  var std1 =  Student();   // 实例化, 调用了类的 默认 构造函数
  std1.id = 24;
  std1.name = "Kevin";
  std1.study();

  var std2 = Student(12, 'yycu');  // 调用有参的构造函数

    var std3 = Student.myCustomConstrunctor();  //调用 自定义的构造函数
}
// 定义类
class Student {
  int id;              // 默认值是null   实例的
  String name;         // 默认值是null   实例的
   
  int age = 32;        // age 是 32     实例的

  Student() {
    // 默认的构造函数      // 可以不用写
    print(" 默认构造函数")
  }

  Student(int id, String name) {    //有参的构造函数
    this.id = id
    this.name = name
  }

  Student(this.id,this.name) {    // 构造函数, 这样写更方便
  }

    
  Student.myCustomConstrunctor() {   // myCustomConstrunctor 这个名字随你起
     // 自定的命名构造函数
     print("自定的命名构造函数");
  }

  void study() {
    int id = 12;   // 作用域就在这个函数里了
    print("${this.name} study");
  }

  void sleep() {
    print("${this.name} sleep");
  }
}
  • 上面没有明显的声明一个 构造函数, 其它是有的,系统提供了一个无参的构造函数 Student()
  • 只能定义一个构造函数, 上面只是演示, java就可以定义多个多种构造函数.
  • 类同名的构造函数, 有参或无参,只能存在一个
  • 命名构造函数,可以多个, 可以和
